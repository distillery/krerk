# handelsbeurs

## SETUP MODE

```
<Enter>: activate setup mode
<azerqsdfwxcv>: select plexi
```

When plexi has been selected, press

```
<arrow up> / <arrow down> <arrow left>  / <arrow right> to move the plexi
```

or press 
```
<o> to resize plexi (make bigger)
<l> make smaller
```
